Detailed Description

template<class T, class P1>
class ArConstFunctor1C< T, P1 >

Functor for a const member function with 1 parameter.

This is a class for const member functions which take 1 parameter. This class contains the knowledge on how to call a const member function on a particular instance of a class. This class should be instantiated by code that wishes to pass off a functor to another piece of code.

For an overall description of functors, see ArFunctor.
