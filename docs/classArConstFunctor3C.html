Detailed Description

template<class T, class P1, class P2, class P3>
class ArConstFunctor3C< T, P1, P2, P3 >

Functor for a const member function with 3 parameters.

This is a class for const member functions which take 3 parameters. This class contains the knowledge on how to call a const member function on a particular instance of a class. This class should be instantiated by code that wishes to pass off a functor to another piece of code.

For an overall description of functors, see ArFunctor.
