Detailed Description

template<class T, class P1, class P2, class P3, class P4, class P5>
class ArFunctor5C< T, P1, P2, P3, P4, P5 >

Functor for a member function with 5 parameters.

This is a class for member functions which take 5 parameters. This class contains the knowledge on how to call a member function on a particular instance of a class. This class should be instantiated by code that wishes to pass off a functor to another piece of code.

For an overall description of functors, see ArFunctor.
