Detailed Description

template<class Ret, class T>
class ArConstRetFunctorC< Ret, T >

Functor for a const member function with return value.

This is a class for const member functions which return a value. This class contains the knowledge on how to call a const member function on a particular instance of a class. This class should be instantiated by code that wishes to pass off a functor to another piece of code.

For an overall description of functors, see ArFunctor.
